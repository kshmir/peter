Política de Privacidad — Peter

Última actualización: 24 de marzo de 2026

Resumen

Peter es un launcher para Android diseñado para adultos mayores. La aplicación no recopila, transmite ni comparte datos personales con terceros. Toda la información se almacena localmente en el dispositivo del usuario.

Datos que almacenamos localmente

  • Lista de aplicaciones permitidas: Los nombres de las aplicaciones que el cuidador ha aprobado para el usuario.
  • Contactos seleccionados: Nombres y números de teléfono de contactos elegidos por el cuidador para mostrar en la pantalla principal. Estos datos provienen del permiso READ_CONTACTS y se almacenan localmente.
  • PIN de administrador: Un hash SHA-256 del PIN. El PIN original nunca se almacena.
  • Preferencias de pantalla: Tamaño de texto y configuración de la cuadrícula.
  • Registro de protección: Eventos de seguridad como instalaciones bloqueadas y notificaciones filtradas, almacenados localmente para revisión del cuidador.

Permisos

  • READ_CONTACTS: Para mostrar los contactos seleccionados por el cuidador en la pantalla de contactos. No se comparten con terceros.
  • CALL_PHONE: Para permitir al usuario llamar directamente a sus contactos de confianza.
  • WRITE_SETTINGS: Para ajustar el tamaño de texto del sistema según las necesidades del usuario.
  • Servicio de Accesibilidad: Para proteger al usuario bloqueando aplicaciones no autorizadas.
  • Acceso a Notificaciones: Para filtrar mensajes sospechosos de WhatsApp y proteger al usuario de estafas.

Compartición de datos

No compartimos ningún dato. Toda la información permanece en el dispositivo. No hay servidores, analíticas, ni conexiones a servicios externos.

Seguridad

El PIN de administrador se almacena como un hash SHA-256. Las comunicaciones de la aplicación no salen del dispositivo.

Privacy Policy — Peter

Last updated: March 24, 2026

Summary

Peter is an Android launcher designed for seniors. The app does not collect, transmit, or share personal data with third parties. All information is stored locally on the user's device.

Data stored locally

  • Allowed apps list: Names of apps the caregiver has approved.
  • Selected contacts: Names and phone numbers chosen by the caregiver. This data comes from the READ_CONTACTS permission and is stored locally only.
  • Admin PIN: A SHA-256 hash. The original PIN is never stored.
  • Display preferences: Text size and grid settings.
  • Protection log: Security events like blocked installs and filtered notifications, stored locally for caregiver review.

Permissions

  • READ_CONTACTS: To display caregiver-selected contacts. Not shared with third parties.
  • CALL_PHONE: To let the user directly call trusted contacts.
  • WRITE_SETTINGS: To adjust system text size for the user's needs.
  • Accessibility Service: To protect the user by blocking unauthorized apps.
  • Notification Access: To filter suspicious WhatsApp messages and protect against scams.

Data sharing

We do not share any data. All information stays on the device. There are no servers, analytics, or external service connections.
